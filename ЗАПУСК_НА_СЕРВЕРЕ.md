# Инструкция по запуску MaxFlash на сервере

## 1. Подключение к серверу

Подключитесь к серверу по SSH:

```bash
ssh devyjones@192.168.0.204
```
Пароль: `19Maxon91!`

## 2. Переход в папку проекта

```bash
cd MaxFlash
```

## 3. Активация виртуального окружения

Перед запуском любых команд необходимо активировать виртуальное окружение:

```bash
source venv/bin/activate
```
После активации в начале строки появится надпись `(venv)`.

## 4. Запуск приложения

### Запуск API сервера
```bash
python run.py
```

### Запуск в фоновом режиме (через screen или nohup)
Если вы хотите, чтобы приложение работало после закрытия терминала, используйте `nohup`:

```bash
nohup python run.py > app.log 2>&1 &
```

Проверить логи:
```bash
tail -f app.log
```

## 5. Обновление кода

Если вы внесли изменения локально и хотите обновить код на сервере:

1. Сделайте `git push` на локальной машине.
2. На сервере выполните:
```bash
git pull
```

## 6. Установка новых зависимостей

Если добавились новые библиотеки:
```bash
pip install -r requirements.txt
```
