────────────────────────────────────

1. Дважды кликни на: ЗАПУСК_БЕЗ_ОКНА.bat
   (Запустится без видимого окна)

2. ИЛИ дважды кликни на: start_hidden.vbs

   (VBS скрипт для скрытого запуска)

ОСТАНОВКА:
───────────

1. Открой Диспетчер задач (Ctrl+Shift+Esc)
2. Найди процесс "pythonw.exe" или "python.exe"
3. Заверши процесс

ИЛИ используй: stop_dashboard.bat

ЕСЛИ НЕ РАБОТАЕТ:
─────────────────

1. Убедись, что установлен Python 3.10+
2. Установи зависимости: pip install -r requirements.txt
3. Используй обычный запуск: python start_final.py
