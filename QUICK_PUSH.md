# Быстрая инструкция по загрузке на GitHub

## Шаг 1: Создайте репозиторий на GitHub

1. Перейдите на https://github.com/devyjones991-ai
2. Нажмите кнопку **"+"** → **"New repository"**
3. Заполните:
   - **Repository name**: `MaxFlash`
   - **Description**: "Integrated Crypto Trading System with Smart Money Concepts, Footprint, Volume Profile, Market Profile and TPO"
   - Выберите **Public** или **Private**
   - **НЕ** добавляйте README, .gitignore или лицензию (они уже есть)
4. Нажмите **"Create repository"**

## Шаг 2: Загрузите код

Remote уже настроен! Просто выполните:

```bash
git push -u origin main
```

Или используйте скрипт:

**Windows:**
```bash
PUSH_TO_GITHUB.bat
```

**Linux/Mac:**
```bash
chmod +x PUSH_TO_GITHUB.sh
./PUSH_TO_GITHUB.sh
```
